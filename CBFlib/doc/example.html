Examples

The following are examples of CBF and imgCIF files prodiced by CBFlib. All of these files were produced from the same MAR image file example.mar2300 provided by Paul Ellis. Thanks to Bob Sweet at BNL for providing the computer access and disk space to store earlier versions of these examples.

The first example, makecbf.cbf (3.7 MB), is a CBF produced by Paul Ellis's test program makecbf. The binary data is CBF_PACKED compressed and carried as raw binary data. The loop for the binary data begins: 

loop_
_array_data.array_id
_array_data.binary_id
_array_data.data
 image_1 1
;
--CIF-BINARY-FORMAT-SECTION--
Content-Type: application/octet-stream;
     conversions="x-CBF_PACKED"
Content-Transfer-Encoding: BINARY
X-Binary-Size: 3745758
X-Binary-ID: 1
Content-MD5: 1zsJjWPfol2GYl2V+QSXrw==

loop_

_array_data.array_id
_array_data.binary_id
_array_data.data
